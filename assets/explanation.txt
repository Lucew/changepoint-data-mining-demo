This tool enables the user to compare change coupling of multiple signals. In conclusion, this tool uses changepoint detection algorithms to compute the changepoint scores for each signal in a collection of signals individually (the scores are normalized by a sliding z-score).

Using all changescores, the tool then computes a reference changescore by computing the mean over all individual scores and computes the difference of the reference score to every score. The hypothesis is that when one signal changes very differently to the reference signal, this isolated change hints at an anomaly. This is especially true if one expects the signals to be coupled (e.g., as part of one machine with input/output relations).

This absolute difference of scores from the mean score is then plotted in the heatmap below.

Possible interactions:
- Select the order of signals in the heatmap by dragging and dropping in the next tile
- Select the signals to be plotted in the heatmap by dragging and dropping in the next tile
- Select signals and time windows by putting shapes into the heatmap
    - These shapes will then create new tiles that plot the selected signals in the specified time window
    - The tile can be navigated to by pressing their corresponding selection number on the keyboard
- Navigate the page by using keyboard numbers [0] will bring you back to the top
